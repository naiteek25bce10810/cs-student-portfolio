# cs-student-portfolio

This is a portfolio website for a Computer Science student, showcasing projects, a resume, and a contact form. The website is built using React and TypeScript, providing a modern and responsive user experience.

## Project Structure

```
cs-student-portfolio
├── src
│   ├── main.tsx          # Entry point of the React application
│   ├── App.tsx           # Main App component with routing
│   ├── components         # Reusable components
│   │   ├── Header.tsx    # Navigation header
│   │   ├── Footer.tsx    # Footer with copyright info
│   │   ├── ProjectCard.tsx# Component to display project details
│   │   └── ContactForm.tsx# Form for visitor inquiries
│   ├── pages             # Different pages of the portfolio
│   │   ├── Home.tsx      # Landing page
│   │   ├── Projects.tsx   # Page listing all projects
│   │   └── Resume.tsx    # Page displaying the resume
│   ├── styles            # CSS styles
│   │   └── globals.css    # Global styles
│   └── data              # Data files
│       └── projects.ts    # Project data
├── index.html            # Main HTML template
├── package.json          # npm configuration
├── tsconfig.json         # TypeScript configuration
├── vite.config.ts        # Vite configuration
├── .gitignore            # Git ignore file
└── README.md             # Project documentation
```

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd cs-student-portfolio
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the development server**:
   ```
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the portfolio.

## Features

- **Responsive Design**: The portfolio is designed to be mobile-friendly and responsive.
- **Project Showcase**: Display your projects with descriptions and links.
- **Contact Form**: Allow visitors to reach out with inquiries.
- **Resume Page**: Present your resume or CV in a clear format.
